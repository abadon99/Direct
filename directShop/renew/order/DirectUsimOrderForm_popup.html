만 29세 이하 고객의 경우 선택하신 요금제 월정액에서 월 5,500원 자동 할인 적용되며, 할인금액은 요금 청구서에서 확인하실 수 있습니다.

데이터쉐어링 가입 전 유의사항
  • 모회선이 5G 요금제인 경우, 5G 쉐어링요금제만 가입이 가능하며 회선당 5천원 요금이 부과됩니다.
    (일부 할인 가능 요금제의 경우 1회선 할인 가능)
  • 모회선이 LTE 요금제인 경우, 최대 2대까지 무료 이용가능하며, 3번째 기기부터 1대 당 8,250원 요금이 부과됩니다.
  • 가입 가능단말: 스마트폰, 패드(음성 패드 포함), 모뎀류, 스마트 기기류 (5G/LTE 단말만 가능)
  • 가입 불가 단말: 워치류(갤럭시 워치, 애플워치 등), 에그(와이브로 모듈 미탑재된 에그 포함),
    라우터, 3G 단말을 포함한 위 가입 가능 단말에 해당되지 않는 단말기 가입 불가
  • 모회선의 납부자와 데이터쉐어링 신청 명의자가 불일치할 경우 개통이 불가합니다.
  • 미성년자 및 법인고객의 경우 개통이 불가합니다.
  • 타사포함 통신요금 미납 시 개통이 불가합니다.
  • 개인별 가입 가능 회선 초과 시 개통이 불가합니다.(개인 신용도에 따라 상이, 최대 5회선)
  • 신용 조회 후 가입 불가 확인 시 개통이 불가합니다.
  • 모회선이 요금제 변경 예약 중일 경우 쉐어링 개통 후 결합이 불가하므로, 예약 해제 후 개통이 가능합니다.
  • 모회선에서 이용 중인 요금제에 따라 데이터쉐어링 서비스 가입이 불가할 수 있습니다. (청소년 요금제, 음성 요금제 등)
  • 쉐어링 개통 후 최초 인식된 서브 단말기 기준으로 사용이 가능하며, 등록 후 다른 종류의 단말기로 유심이동은 불가합니다.
  • 개인 보안등급 설정이 되어있는 경우 개통이 불가합니다.
데이터쉐어링 가입 유의사항
5G 데이터 쉐어링
  • 가입할 회선이 5G 요금제인 경우만 가입 가능합니다.
  • 가입할 회선 요금제가 데이터 무제한 요금제일 경우 각 요금제의 ‘스마트기기/테더링 이용 한도’ 내에서 공유 사용 가능하며 (예. 슈퍼플랜 프리미엄 초이스 100GB / 스페셜 초이스 70GB / 스페셜 50GB / 베이직 초이스 40GB / 베이직 20GB), 데이터 기본 제공량이 존재하는 요금제의 경우 기본 데이터 제공량 내에서 공유 가능합니다. 공유 데이터 소진 후 최대 200Kbps 속도제어로 사용 가능합니다.
  • 음성/영상통화(데이터를 이용한 통화 포함), 문자(MMS/LMS/SMS), 채팅+(채팅플러스) 서비스는 사용 불가합니다.
  • 공유 데이터는 국내 사용만 가능하며, 쉐어링 된 단말기로 해외 로밍 이용 시는 각 국가별 별도 계약한 요율이 적용 됩니다.
  • 5G 요금제 가입자(가입할 회선)에 동일 명의에 한하여 최대 9회선까지 가입 가능합니다. (5G데이터쉐어링 가입 시 고객별 이동전화 개통 가능 회선수에 포함)
  • 가입할 회선의 납부자와 데이터쉐어링 명의자가 다를 경우 데이터쉐어링 개통이 불가능합니다.
  • 쉐어링 서비스 이용 단말기로 타 기기의 모뎀역할을 하거나 테더링으로 사용할 수 없습니다.
  • 상업적인 목적으로 이용할 수 없으며, 고객의 불/편법적인 서비스 이용이 확인되거나 추정 될 경우 해당 서비스 이용을 제한할 수 있습니다. (예. 금전 등 부정한 이익을 얻을 목적으로 해당 서비스를 이용하는 행위, 제3자에게 임의로 해당 서비스를 임대하는 행위, 기타 사회 통념상 불/편법적인 행위로 인정되는 경우 등)
LTE 데이터쉐어링
  • LTE 스마트폰에서 제공하는 데이터량을 스마트기기(스마트폰, 태블릿PC, 패드, 스마트카메라 등)와 공유할 수 있습니다.
    • 3G 스마트기기에서도 이용 가능합니다.
    • 최대 2대까지 무료로 이용할 수 있고, 3번째 기기 부터 요금이 부과됩니다.
  • 이용요금은 각 스마트기기별로 부과하며, 공유 가능한 최대 스마트기기 수는 9대입니다.
    • 고객에 따라 신청(추가) 가능한 최대 기기 수가 다를 수 있습니다.
  • LTE데이터의 이월된 데이터는 LTE 데이터쉐어링에서 사용 가능합니다.
  • LTE 요금제의 기본제공량까지 데이터쉐어링에서 이용 가능하며, 기본 제공 데이터 모두 소진 시 최대 200Kbps 속도로 지속 이용 가능합니다.
  • 기본제공 데이터는 국내통화료만 포함하며, 로밍 이용 시 각 국가별 별도 계약한 요율이 적용됩니다.
    • 로밍 시 LTE 데이터쉐어링 이용이 불가하며, 공유하는 스마트기기 로밍 시 각 국가별 요율에 따라 요금이 부과됩니다.
  • 와이브로 기능이 있는 모뎀에서는 사용할 수 없습니다.
  • 공유하는 스마트기기는 LTE 휴대전화와 동일한 명의로 개통되어야 합니다.
  • 가입할 회선의 납부자와 데이터쉐어링 명의자가 다를 경우 데이터쉐어링 개통이 불가능합니다.
  • LTE WARP 알 요금제, 선불 등 지능망 요금제 가입자는 LTE 데이터쉐어링을 이용할 수 없습니다.
  • LTE 휴대전화 해지 또는 LTE 데이터쉐어링 해지 시, LTE 데이터쉐어링으로 개통된 스마트기기는 직권 해지되며, 쉐어링 단말은 일시금 납부됩니다.
  • LTE 휴대전화 일시 정지 시, LTE 데이터쉐어링으로 개통된 스마트기기도 일시 정지됩니다.
  • LTE 데이터쉐어링으로 개통된 스마트기기는 다른 데이터전용 요금제로 가입이 불가합니다.
  • LTE 데이터쉐어링 이용 기기에서는 mVoIP 및 테더링이 불가합니다.
  • LTE 요금제 중 안심차단형 요금제의 경우, 데이터 안심차단 시 스마트기기에서도 데이터를 이용할 수 없습니다. (단, 데이터 추가 충전 시에는 사용 가능합니다.)
  • 본 요금제를 통한 음성/영상통화, 문자메세지는 이용이 불가합니다.
  • 스마트기기(태블릿PC, 패드 등)을 해외에서 분실한 경우, USIM 카드 도용으로 인해 과다한 해외통화 요금이 발생할 수 있으니, 로밍센터(+82-2-2190-0901)를 통해 반드시 분실신고를 해주시기 바랍니다
