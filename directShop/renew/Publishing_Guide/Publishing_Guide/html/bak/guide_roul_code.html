코드 규칙사항

HTML

General Formatting Rules

  • 문서의 기본구조인 html5의 규격을 최대한 준수하며, 의미에 맞게 시멘틱 마크업을 한다.
  • 들여쓰기, 빈줄삽입, 주석 등은 아래 형태로 진행한다.

들여쓰기

  • 코드 및 단락의 들여쓰기는 1탭 들여쓰기를 하며, 1탭의 크기는 4칸으로 설정한다.

빈줄

  • 문서의 가독성 및 단락 구분을 위하여 필요시 빈줄을 추가한다.
  • 단 빈줄 추가는 최대 1줄을 초과하지 아니한다.

주석

  • 문서의 주석 표시는 문서를 열람하는 모든이가 단락의 구분을 확인할 수 있도록 제공을 원칙으로 한다.
  • html문서의 주석은 시작주석과 끝주석을 표기하여 단락의 구분을 명확히 할 수 있도록 한다.
  • 시작주석은 <!-- 내용 -->으로 하며, 끝주석은 <!-- //내용 --> 으로 한다.
								<!-- Header -->
								<header id="header">
									<!-- Gnb -->
									<nav class="gnb">
										.....
									</nav>
									<!-- //Gnb -->
								</header>
								<!-- //Header -->

속성 선언 순서

  • 태그의 속성은 아래 우선순위에 맞게 코딩을 한다.
HTML 속성선언순서
구분 속성 속성 값
1. 기능

href, src, type
2. 이름

name, id
3. 정보

value, title, alt
4. 옵션

target, tabindex
5. 꾸미기

class

 
								<a href="#link" title="새창" target="" class="">링크</a>
								<iframe src="링크" title="빈프레임"></iframe>
								<img src="링크" alt="대체텍스트" />
								<input type="text" name="" id="" value="" title="" class="" style="" disabled />

CSS

General Formatting Rules

  • 속성값은 기본은 작은따옴표 ''를 사용한다.
  • css문서는 기본으로 들여쓰기는 사용하지 않는다. (단 미디어쿼리의 사용으로 인하여 코드구분을 하기위하여 1탭 4칸 들여쓰기는 허용한다.)
  • 속성값을 적용하기 위하여 id, class명 과 {} 사이에는 한칸 띄어쓰기를 한다.
  • css의 끝주석은 작성하지 않는다.
  • 문서의 가독성 및 단락 구분을 위하여 필요시 빈줄을 추가한다.
  • 단 빈줄 추가는 최대 1줄을 초과하지 아니한다.

속성값은 작은따옴표('') 사용

  • 폰트, 필터, content등 속성에 사용되는 값은 작은따옴표('')를 사용한다.
								.class {font-family:'dotum';}
								.filter:progid:DXImageTransform.Microsoft.AlphaImageLoader(src='img.jpg', sizingMethod='scale';}
								.class:after{content:'.'}
								.class {background:url(img.gif) no-repeat}

세미콜론 ;

  • 속성 구분을 위한 ;은 마지막에도 사용을 한다.
								.class {margin:0; padding:0;}

공백

  • 선택자와 속성값을 지정하는 한칸의 공백을 사용한다.
  • 속성값을 입력하는 {}사이 에는 공백을 사용하지 않는다.
								.class {margin:0; padding:0;}

주석 / 빈줄

  • css는 시작주석만 표시를 하고 종료 주석은 작성하지 않는다.
  • 그룹간의 구분을 위한 빈줄은 1줄을 초과하지 않늗다.
								/* Header */
								#header {}

								/* gnb */
								.gnb {}
								.gnb li {}
								.gnb li a {}

								/* container */
								#container {}

공통선택자(*)

  • 공통선택자인 *의 사용은 웹페이지의 성능 저하를 야기할수 있으므로 되도록이면 사용을 하지 않는다.
								* {margin:0; padding:0;}
