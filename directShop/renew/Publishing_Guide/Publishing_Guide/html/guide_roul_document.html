문서 구조 가이드

공통 사항

							<!DOCTYPE html>
							<html lang="ko">
							<head>
								as-is

								<!-- ktd : 추가 css -->
								<link rel="stylesheet" href="/directShop/css/ktDirect.css">

								<!-- ktd : 추가 js -->
								<script src="/directShop/js/ui/pub_ktDirect.js" type="text/javascript"></script>
							</head>
							<body>
								<!-- ktd : Direct 안내 -->
								<div class="ktd-directInfo">
									Direct 안내 신규 영역
								</div>
								<!-- //ktd : Direct 안내 -->

								<!-- ktd : 사이트 영역 -->
								<div class="ktd-conWrap">
									기존 as-is 소스
								</div>
								<!-- //ktd : 사이트 영역 -->
							</body>
							</html>

공통 적용 사항

  • "head" 영역 제일 하단으로 "ktDirect.css, pub_ktDirect.js" 추가
  • "body" 에 추가되는 "ktd-wrap"은 상기 pub_ktDirect.js에서 적용됨(별도 추가 작업 없음)
  • 모든 페이지(새창등 별도페이지 제외) 상기 공통사항 예시처럼
    <div class="ktd-directInfo">
        Direct 안내 신규 영역
    </div>
    <div class="ktd-conWrap">
        기존 as-is 소스
    </div>
    "다이렉트 안내 영역"과 기존 as-is가 표시될 영역이 구분 되어야 함

헤더 영역 : header

가입혜택 영역 : div class="ktd-gift-section"
